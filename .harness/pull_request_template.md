## Comment Triggers

<details>
<summary>Use the following comments to re-trigger PR Checks</summary>

- GitLeaks: `trigger gitLeaks`
- MessageMetadata: `trigger messagecheck`
- Helm Unit Tests: `trigger helmtests`
- semgrepSastPr: `trigger semgrep`

</details>
