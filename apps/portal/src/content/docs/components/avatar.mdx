The Avatar component provides a way to represent an image with a fallback for when the image cannot be loaded. It also supports displaying initials derived from the `name` prop.

## Usage

### Image (Default)

```typescript jsx
<Avatar name="John Doe" src="https://github.com/shadcn.png" />
<Avatar name="John Doe" src="https://github.com/shadcn.png" size="sm" />
<Avatar name="John Doe" src="https://github.com/shadcn.png" size="lg" />
```

### Image (Rounded)

```typescript jsx
<Avatar name="John Doe" src="https://github.com/shadcn.png" rounded />
<Avatar name="John Doe" src="https://github.com/shadcn.png" size="sm" rounded />
<Avatar name="John Doe" src="https://github.com/shadcn.png" size="lg" rounded />
```

### Initials (Default)

```typescript jsx
<Avatar name="John Doe" />
<Avatar name="John Doe" size="sm" />
<Avatar name="John Doe" size="lg" />
```

### Initials (Rounded)

```typescript jsx
<Avatar name="John Doe" rounded />
<Avatar name="John Doe" size="sm" rounded />
<Avatar name="John Doe" size="lg" rounded />
```

### Avatar Icon (Default)

```typescript jsx
<Avatar />
<Avatar size="sm" />
<Avatar size="lg" />
```

### Avatar Icon (Rounded)

```typescript jsx
<Avatar rounded />
<Avatar size="sm" rounded />
<Avatar size="lg" rounded />
```

## API Reference

```typescript jsx
<Avatar
  name="John Doe"                     // [OPTIONAL] Full name of the subject
  src="https://github.com/shadcn.png" // [OPTIONAL] Source URL of the image
  size="lg"                           // [OPTIONAL] Size of the avatar ('default' | 'sm' | 'lg')
  rounded                             // [OPTIONAL] Whether the avatar is circular
/>
```
