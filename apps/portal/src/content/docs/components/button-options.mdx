The `ButtonWithOptions` component provides a flexible UI element for displaying a button with a dropdown of options. It can behave as a radio button group if a `selectedValue` is provided, or as a regular dropdown otherwise.

## Usage

```typescript jsx
import { ButtonWithOptions } from '@harnessio/ui/components'

return (
  <div className="space-y-4">
    <div className="mb-4 text-white">
      <div className="text-lg font-bold">Primary Button (Default Size)</div>
      <ButtonWithOptions
        id="primary-button-default"
        handleButtonClick={() => alert('Primary Button clicked')}
        options={[
          { value: 'option1', label: 'Option 1', description: 'Description for option 1' },
          { value: 'option2', label: 'Option 2', description: 'Description for option 2' },
        ]}
        handleOptionChange={(val) => alert(\`Selected: \${val}\`)}
        theme="primary"
      >
        Primary Button
      </ButtonWithOptions>
    </div>
  </div>
)
```

## Anatomy

All parts of the ButtonWithOptions component can be imported and composed as required.

```typescript jsx
<ButtonWithOptions
  id="example-button-with-options"
  handleButtonClick={() => alert('Button clicked')}
  options={[
    { value: 'option1', label: 'Option 1', description: 'Description for option 1' },
    { value: 'option2', label: 'Option 2', description: 'Description for option 2' },
  ]}
  handleOptionChange={(val) => alert(\`Selected: \${val}\`)}
>
  Button Text
</ButtonWithOptions>
```
