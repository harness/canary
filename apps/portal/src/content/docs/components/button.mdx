The `Button` component allows you to display a button the user can use to trigger actions or submit forms.

## Usage

```typescript jsx
import { Button } from '@harnessio/ui/components'

//...

return (
  <Button variant="destructive" size="lg" onClick={handleDelete}>
    Delete
  </Button>
)
```
