The `CardSelect` component provides an accessible and customizable card-based selection interface. It supports both single and multiple selection modes, making it suitable for forms, settings, and decision workflows.

## Usage

```typescript jsx
import { CardSelect } from '@harnessio/ui/components'

// Controlled Single Selection
const [selected, setSelected] = useState('option-1')

return (
  <CardSelect.Root type="single" value={selected} onValueChange={setSelected}>
    <CardSelect.Item value="option-1" icon="branch">
      <CardSelect.Title>Controlled Option</CardSelect.Title>
      <CardSelect.Description>External state management</CardSelect.Description>
    </CardSelect.Item>
  </CardSelect.Root>
)

// Uncontrolled Multiple Selection
return (
  <CardSelect.Root
    type="multiple"
    defaultValue={['one']}
    onValueChange={(vals) => console.log('Selected:', vals)}
  >
    <CardSelect.Item value="one" icon="branch">
      <CardSelect.Title>Uncontrolled Option</CardSelect.Title>
      <CardSelect.Description>Internal state management</CardSelect.Description>
    </CardSelect.Item>
  </CardSelect.Root>
)
```

## Anatomy

The `CardSelect` component is composed of the following parts:

```typescript jsx
<CardSelect.Root>
  <CardSelect.Item>
    <CardSelect.Title />
    <CardSelect.Description />
  </CardSelect.Item>
</CardSelect.Root>
```

## Selection Modes

The `CardSelect` supports two selection types:

- `single`: Behaves like a radio group, allowing only one selection at a time
- `multiple`: Behaves like a checkbox group, allowing multiple selections
