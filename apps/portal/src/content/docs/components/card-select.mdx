The `CardSelect` component provides an accessible and customizable card-based selection interface. It supports both single and multiple selection modes, making it suitable for forms, settings, and decision workflows.

import { DocsPage } from "@/components/docs-page";

## Usage

```tsx
import { CardSelect } from "@harnessio/ui";

<CardSelect.Root type="multiple" value={["one"]} onValueChange={setValues}>
  <CardSelect.Item value="one" icon="plus">
    <CardSelect.Title>Choice A</CardSelect.Title>
    <CardSelect.Description>Option with an icon</CardSelect.Description>
  </CardSelect.Item>

  <CardSelect.Item value="two" logo="logo-somebrand">
    <CardSelect.Title>Choice B</CardSelect.Title>
    <CardSelect.Description>Option with a logo</CardSelect.Description>
  </CardSelect.Item>
</CardSelect.Root>;
```

## Modes

The `CardSelect` supports two selection types:

- `single`: behaves like a radio group.
- `multiple`: behaves like a checkbox group.
