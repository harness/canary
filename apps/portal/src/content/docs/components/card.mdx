The `Card` component provides a flexible UI element for displaying content within a card layout. It supports various configurations such as sizes, orientations, and the position of the image relative to the content.

## Usage

```typescript jsx
import { Card } from '@harnessio/ui'

// ...

return (
  <Card.Root orientation="vertical" position="start">
    <Card.Image src="/card-image.png" width="100%" height="150px" />
    <Card.Title>Card Title</Card.Title>
    <Card.Content>
      This is the card content. The image is displayed above the content.
    </Card.Content>
  </Card.Root>
)
```

## Position

The `Card` component supports a `position` prop to control the placement of the image relative to the content:

- **Vertical Orientation**: Use `position="start"` to display the image above the content, or `position="end"` to display it below.
- **Horizontal Orientation**: Use `position="start"` to display the image to the left of the content, or `position="end"` to display it to the right.

## Anatomy

All parts of the `Card` component can be imported and composed as required.

```typescript jsx
<Card.Root>
  <Card.Header>
    <Card.Title />
    <Card.Description />
  </Card.Header>
  <Card.Content />
  <Card.Footer />
</Card.Root>
```
