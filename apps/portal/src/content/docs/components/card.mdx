The `Card` component provides a flexible UI element for displaying content within a card layout. It supports various configurations such as sizes, orientations, and states.

import { DocsPage } from "../../../components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex flex-col gap-10">
    <div>
      <h6 className="font-semibold mb-2 text-center">Basic Card</h6>
      <Card.Root>
        <Card.Image src="/card-image.png" width="100%" height="150px" />
        <Card.Title>Card Title</Card.Title>
        <Card.Content>
          This is the card content. You can customize it as needed.
        </Card.Content>

          <Card.Title>Horizontal Card</Card.Title>

## Usage

```typescript jsx
import { Card } from '@harnessio/ui'

// ...

return (
  <Card.Root size="md">
    <Card.Image src="/card-image.png" width="100%" height="150px" />
    <Card.Title>Card Title</Card.Title>
    <Card.Content>
      This is the card content. You can customize it as needed.
    </Card.Content>
  </Card.Root>
)
```

## Sizes

The `Card` component supports three sizes: `sm`, `md`, and `lg`.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex items-center gap-6">
    <Card.Root size="sm">
      <Card.Title>Small Card</Card.Title>
    </Card.Root>
    <Card.Root size="md">
      <Card.Title>Medium Card</Card.Title>
    </Card.Root>
    <Card.Root size="lg">
      <Card.Title>Large Card</Card.Title>
    </Card.Root>
</div>`}
/>
