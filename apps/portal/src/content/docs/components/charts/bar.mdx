The Bar chart renders horizontal bars using the `CartesianChart` component with
a series `type` of `CartesianSeriesVisualizationType.Bar`. Because the value runs
along the X axis and the categories sit on the Y axis, it is well suited to
ranking many labelled items — for example languages by adoption, repositories by
contribution, or teams by throughput.

## Horizontal bars

For a horizontal bar chart, put the numeric measure on the **X** axis
(`ScaleType.Linear`) and the categories on the **Y** axis (`ScaleType.Band`).
Each data point is an `{ x, y }` pair where `x` is the numeric value and `y` is
the category label.

<DocsPage.ChartExample
  client:only
  code={`import {
  AxisLocation,
  AxisType,
  CartesianChart,
  CartesianSeriesVisualizationType,
  LegendPosition,
  PopoverServiceProvider,
  ScaleType,
} from "@harness/dashboard-visualization";
import "@harness/dashboard-visualization/style.css";

const bars = [
  { label: "Python", value: 89.9, color: "#3b82f6" },
  { label: "TypeScript", value: 72.1, color: "#6366f1" },
  { label: "Go", value: 45.0, color: "#22c55e" },
];

const series = [
  {
    name: "ai-written-pct",
    displayName: "AI Written %",
    color: bars[0].color,
    type: CartesianSeriesVisualizationType.Bar,
    getColor: (datum) =>
      bars.find((b) => b.label === datum?.y)?.color ?? bars[0].color,
    data: bars.map((item) => ({ x: item.value, y: item.label })),
  },
];

const xAxis = {
  type: AxisType.X,
  location: AxisLocation.Bottom,
  scale: ScaleType.Linear,
  gridLines: true,
  tickLabels: true,
  min: 0,
  max: 100,
};

const yAxis = {
  type: AxisType.Y,
  location: AxisLocation.Left,
  scale: ScaleType.Band,
  gridLines: false,
  tickLabels: true,
  size: 90,
  getLabel: (value) => String(value),
};

render(
  <PopoverServiceProvider>
    <div style={{ width: 480, height: 280 }}>
      <CartesianChart
        series={series}
        xAxisOption={xAxis}
        yAxisOption={yAxis}
        showXAxis
        showYAxis
        padding={{ top: 0, left: 0, bottom: 0, right: 16 }}
        legend={LegendPosition.None}
        enableTooltips
      />
    </div>
  </PopoverServiceProvider>
);`}
/>

## Usage

```tsx
import {
  AxisLocation,
  AxisType,
  CartesianChart,
  CartesianSeriesVisualizationType,
  PopoverServiceProvider,
  ScaleType,
} from "@harness/dashboard-visualization";
import "@harness/dashboard-visualization/style.css";

// A bar series is a Series whose `type` is Bar. Put the value on X (Linear) and
// the category label on Y (Band).
<PopoverServiceProvider>
  <CartesianChart
    series={[
      {
        name: "series-1",
        color: "#3b82f6",
        type: CartesianSeriesVisualizationType.Bar,
        data: [{ x: 73.5, y: "TypeScript" }],
      },
    ]}
    xAxisOption={{
      type: AxisType.X,
      location: AxisLocation.Bottom,
      scale: ScaleType.Linear,
      min: 0,
    }}
    yAxisOption={{
      type: AxisType.Y,
      location: AxisLocation.Left,
      scale: ScaleType.Band,
    }}
  />
</PopoverServiceProvider>;
```

## Props

The Bar chart uses the same `CartesianChart` props as the Column chart; only the
series `type` and the axis scale orientation differ. The key fields are below.

## Guidelines

<Aside type="tip">
  Sort bars by value (descending) so the ranking reads top-to-bottom, and give
  the category axis enough `size` to avoid truncating labels.
</Aside>

<Aside>
  Use the horizontal Bar chart when category labels are long or when there are
  many categories to rank. For time-based comparisons use the vertical Column
  chart instead.
</Aside>
