---
title: Dropdown Menu
description: A vertically stacked menu that appears when triggered by a button.
---

The Dropdown Menu component provides a flexible menu interface that appears when triggered by a button. It's built on top of Radix UI's dropdown menu primitive and includes various subcomponents for creating rich menu structures.

## Usage

```typescript jsx
import { DropdownMenu } from '@harnessio/ui/components'

return (
  <DropdownMenu.Root>
    <DropdownMenu.Trigger>
      <Button>
        Options
        <Icon name="chevron-down" className="ml-2 h-4 w-4" />
      </Button>
    </DropdownMenu.Trigger>
    <DropdownMenu.Content>
      <DropdownMenu.Item>New File</DropdownMenu.Item>
      <DropdownMenu.Item>Open</DropdownMenu.Item>
      <DropdownMenu.Separator />
      <DropdownMenu.Sub>
        <DropdownMenu.SubTrigger>
          More Options
          <Icon name="chevron-right" className="ml-auto h-4 w-4" />
        </DropdownMenu.SubTrigger>
        <DropdownMenu.SubContent>
          <DropdownMenu.Item>Save As...</DropdownMenu.Item>
          <DropdownMenu.Item>Export</DropdownMenu.Item>
        </DropdownMenu.SubContent>
      </DropdownMenu.Sub>
      <DropdownMenu.Separator />
      <DropdownMenu.CheckboxItem checked={true}>
        Show Line Numbers
      </DropdownMenu.CheckboxItem>
      <DropdownMenu.CheckboxItem checked={false}>
        Word Wrap
      </DropdownMenu.CheckboxItem>
      <DropdownMenu.Separator />
      <DropdownMenu.RadioGroup value="vscode">
        <DropdownMenu.RadioItem value="vscode">
          VS Code Theme
        </DropdownMenu.RadioItem>
        <DropdownMenu.RadioItem value="atom">
          Atom Theme
        </DropdownMenu.RadioItem>
        <DropdownMenu.RadioItem value="sublime">
          Sublime Theme
        </DropdownMenu.RadioItem>
      </DropdownMenu.RadioGroup>
    </DropdownMenu.Content>
  </DropdownMenu.Root>
)
```

## Anatomy

The Dropdown Menu is composed of several subcomponents that can be imported and used as needed:

```typescript jsx
<DropdownMenu.Root>
  <DropdownMenu.Trigger />
  <DropdownMenu.Content>
    <DropdownMenu.Item />
    <DropdownMenu.Group>
      <DropdownMenu.Item />
    </DropdownMenu.Group>
    <DropdownMenu.Separator />
    <DropdownMenu.Sub>
      <DropdownMenu.SubTrigger />
      <DropdownMenu.SubContent />
    </DropdownMenu.Sub>
    <DropdownMenu.CheckboxItem />
    <DropdownMenu.RadioGroup>
      <DropdownMenu.RadioItem />
    </DropdownMenu.RadioGroup>
  </DropdownMenu.Content>
</DropdownMenu.Root>
```
