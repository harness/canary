The `StatusBadge` component is used to highlight and display small pieces of information, such as status indicators or labels. It supports multiple visual variants and color themes, and is non-interactive (read-only).

## Usage

```typescript jsx
import { StatusBadge } from '@harnessio/ui/components'

// ...

// Status badge with pulse
<StatusBadge variant="status" theme="success" pulse>Status</StatusBadge>

// Status badge with square indicator
<StatusBadge variant="status" theme="success" indicatorShape="square">Status</StatusBadge>

// Other variants (pulse and indicatorShape not allowed)
<StatusBadge variant="primary" theme="info">Info</StatusBadge>
<StatusBadge variant="secondary" theme="warning">Warning</StatusBadge>
<StatusBadge variant="outline" theme="danger">Danger</StatusBadge>
<StatusBadge variant="ghost" theme="muted">Muted</StatusBadge>
```

## Themes

The `StatusBadge` component supports different color themes, including `muted` (default), `success`, `warning`, `danger`, `info`, `merged`, and `risk`.

## Variants

The `StatusBadge` component supports different visual variants, including `primary` (default), `secondary`, `outline`, `ghost`, and `status`.

- `status` variant supports an optional `pulse` prop for animated status indication.
- For all other variants, the `pulse` prop is not allowed.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex flex-wrap gap-cn-sm items-end">
    <StatusBadge variant="primary" theme="muted">Primary</StatusBadge>
    <StatusBadge variant="secondary" theme="muted">Secondary</StatusBadge>
    <StatusBadge variant="outline" theme="muted">Outline</StatusBadge>
    <StatusBadge variant="ghost" theme="muted">Ghost</StatusBadge>
    <StatusBadge variant="status" theme="success">Status</StatusBadge>
</div>`}
/>

## Sizes

The `StatusBadge` component comes in two sizes: `sm` and `md`.

## Icon support

The `StatusBadge` component supports an optional `icon` prop to display an icon inside the badge.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex flex-wrap items-end gap-cn-sm">
    <StatusBadge icon="check" variant="ghost" theme="success">Saved</StatusBadge>

    <StatusBadge icon="info-circle" variant="outline" theme="info">Info</StatusBadge>
</div>`}
/>

## Status StatusBadges

Status badges can optionally include a pulsing indicator to indicate activity using the `pulse` prop.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex flex-wrap items-end gap-cn-sm">
    <StatusBadge variant="status" theme="success">Online</StatusBadge>
    <StatusBadge variant="status" theme="warning">Warning</StatusBadge>
    <StatusBadge variant="status" theme="danger">Error</StatusBadge>
    <StatusBadge variant="status" theme="success" pulse>Live</StatusBadge>
</div>`}
/>

## Indicator Shape

Status badges support different indicator shapes via the `indicatorShape` prop. The default is `circle` (fully rounded), and `square` provides a slightly rounded square indicator.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex flex-wrap items-end gap-cn-sm">
    <StatusBadge variant="status" theme="success">Circle (default)</StatusBadge>
    <StatusBadge variant="status" theme="success" indicatorShape="square">Square</StatusBadge>
    <StatusBadge variant="status" theme="warning" indicatorShape="square" pulse>Square with pulse</StatusBadge>
</div>`}
/>
