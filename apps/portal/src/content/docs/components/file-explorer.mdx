The File Explorer component provides a UI element for browsing and selecting files or directories. It displays a hierarchical view of the file system.

## Usage

```typescript jsx
import { Root, FileItem, FolderItem } from '@harnessio/ui/components/file-explorer'

return (
  <Root value={['src']} onValueChange={(value) => console.log('Value changed:', value)}>
    <FolderItem value="src" link="/src">
      src
      <FileItem link="/src/index.tsx">index.tsx</FileItem>
    </FolderItem>
    <FileItem link="/package.json">package.json</FileItem>
  </Root>
)
```

## Anatomy

The File Explorer component can be used to display a hierarchical view of the file system and handle file or directory selection.

```typescript jsx
<Root value={['src']} onValueChange={(value) => console.log('Value changed:', value)}>
  <FolderItem value="src" link="/src">
    src
    <FileItem link="/src/index.tsx">index.tsx</FileItem>
  </FolderItem>
  <FileItem link="/package.json">package.json</FileItem>
</Root>
```
