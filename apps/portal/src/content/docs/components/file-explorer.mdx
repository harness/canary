The File Explorer component provides a UI element for browsing and selecting files or directories. It displays a hierarchical view of the file system.

## Usage

```typescript jsx
import { FileExplorer } from '@harnessio/ui/components'

return (
  <FileExplorer.Root
    value={['src']}                         // [REQUIRED] Currently selected items
    onValueChange={(value) => console.log('Value changed:', value)} // [REQUIRED] Handler for selection changes
  >
    <FileExplorer.FolderItem
      value="src"                          // [OPTIONAL] Folder identifier
      link="/src"                          // [REQUIRED] Navigation link
      isActive={false}                     // [OPTIONAL] Active state
      chevronClassName="custom-chevron"    // [OPTIONAL] Custom chevron styling
    >
      src
      <FileExplorer.FileItem
        link="/src/index.tsx"              // [OPTIONAL] Navigation link
        isActive={false}                   // [OPTIONAL] Active state
      >
        index.tsx
      </FileExplorer.FileItem>
    </FileExplorer.FolderItem>
    <FileExplorer.FileItem link="/package.json">package.json</FileExplorer.FileItem>
  </FileExplorer.Root>
)
```
