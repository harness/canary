The `Hyperlink` component provides styled links for navigation within and outside the application. It offers two size options and distinguishes between internal and external links with appropriate visual indicator. The component ensures consistent styling while maintaining accessibility standards.

import { DocsPage } from "../../../components/docs-page";

    <div className="flex flex-col gap-4">
        <h5>Standard usage</h5>

        <Hyperlink href="/my-link" data-state="hovered">Internal Link (hovered)</Hyperlink>
        <Hyperlink href="/my-link" data-state="visited">Internal Link</Hyperlink>
    </div>

    <hr />

    <div className="flex flex-col gap-4">

        <h5>Primary variant</h5>

    </div>

</div>`}

## Usage

```typescript jsx
import { Hyperlink } from '@harnessio/ui/components'

// ...

return (
  <>
    <Hyperlink href="/your-url-here">Username</Hyperlink>
  </>
)
```

## API Reference

The `Label` component provides a styled label for form elements with customizable options.

```typescript jsx
<Label
  htmlFor="inputId" // [OPTIONAL] ID of the form control this label is associated with
  optional          // [OPTIONAL] Indicate if the associated form field is optional
>
  Label Text
</Label>
```
