The `IconV2` component allows you to render various icons with customizable properties such as size, color theme, and name. It will soon replace the old `Icon` component.

import { DocsPage } from "../../../components/docs-page";
import IconGrid from "../../../components/IconGrid.astro";

<DocsPage.ComponentExample
  client:only
  code={`<IconV2 name="check" size={48} />`}
/>

## Colored Icon

By default, IconV2 components will inherit the color from the parent. You can also set the color explicitly using the `className` prop.

<DocsPage.ComponentExample
  client:only
  code={`<Layout.Flex gap="md">
    <div className="text-cn-foreground-success">
      <IconV2 name="check" size={32} />
    </div>
    <IconV2 className="text-cn-foreground-danger" name="xmark" size={32} />
  </Layout.Flex>`}
/>

## Usage

```typescript jsx
import { IconV2 } from '@harnessio/ui/components'

//...

return <IconV2 name="check" size={48} />
```

## API Reference

The `IconV2` component can be sized either using the `size` prop.
The `size` prop can be used as a shorthand to set both height and width.

```typescript jsx
<IconV2
  name="double-tick"   // name of the icon
  size={16}            // [OPTIONAL] size of the icon in pixels
  className="my-class" // [OPTIONAL] custom class name
/>
```
