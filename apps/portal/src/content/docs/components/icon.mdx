The `IconV2` component allows you to render various icons with customizable properties such as size, color theme, and name. It will soon replace the old `Icon` component.

import { DocsPage } from "../../../components/docs-page";
import IconGrid from "../../../components/IconGrid.astro";

<DocsPage.ComponentExample
  client:only
  code={`<IconV2 name="check" size="lg" />`}
/>

## Size Variants

IconV2 supports multiple predefined size variants: `2xs`, `xs`, `sm`, `md`, and `lg`. The default size is `sm`.

<DocsPage.ComponentExample
  client:only
  code={`<Layout.Flex gap="md" align="center">
    <IconV2 name="star" size="2xs" />
    <IconV2 name="star" size="xs" />
    <IconV2 name="star" size="sm" />
    <IconV2 name="star" size="md" />
    <IconV2 name="star" size="lg" />
  </Layout.Flex>`}
/>

## Colored Icon

By default, IconV2 components will inherit the color from the parent. You can also set the color explicitly using the `className` prop.

<DocsPage.ComponentExample
  client:only
  code={`<Layout.Flex gap="md">
    <div className="text-cn-foreground-success">
      <IconV2 name="check" size="md" />
    </div>
    <IconV2 className="text-cn-foreground-danger" name="xmark" size="md" />
  </Layout.Flex>`}
/>

## Usage

```typescript jsx
import { IconV2 } from '@harnessio/ui/components'

//...

return <IconV2 name="check" size="lg" />
```

## API Reference

The `IconV2` component can be sized using the `size` prop, which applies predefined size classes.

```typescript jsx
<IconV2
  name="double-tick"   // name of the icon
  size="md"           // [OPTIONAL] size variant (2xs, xs, sm, md, lg)
  className="my-class" // [OPTIONAL] custom class name
  skipSize={false}    // [OPTIONAL] skip size classes if custom styling is applied
/>
```
