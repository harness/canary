The Input OTP component provides a UI element for entering one-time passwords (OTPs). It includes slots for each digit and supports custom separators.

import { DocsPage } from "@/components/docs-page";

  client:only

    <InputOTP.Slot index={2} />
  </InputOTP.Group>
  <InputOTP.Separator />
  <InputOTP.Group>
    <InputOTP.Slot index={3} />
    <InputOTP.Slot index={4} />
    <InputOTP.Slot index={5} />
  </InputOTP.Group>
</InputOTP.Root>`}
/>

## Usage

```typescript jsx
import { InputOTP } from '@harnessio/ui/components'

return (
  <InputOTP.Root maxLength={6}>
    <InputOTP.Group>
      <InputOTP.Slot index={0} />
      <InputOTP.Slot index={1} />
      <InputOTP.Slot index={2} />
    </InputOTP.Group>
    <InputOTP.Separator />
    <InputOTP.Group>
      <InputOTP.Slot index={3} />
      <InputOTP.Slot index={4} />
      <InputOTP.Slot index={5} />
    </InputOTP.Group>
  </InputOTP.Root>
)
```

## Anatomy

The Input OTP component can be used to enter one-time passwords with custom separators.

```typescript jsx
<InputOTP.Root maxLength={6}>
  <InputOTP.Group>
    <InputOTP.Slot index={0} />
    <InputOTP.Slot index={1} />
    <InputOTP.Slot index={2} />
    <InputOTP.Separator />
    <InputOTP.Slot index={3} />
    <InputOTP.Slot index={4} />
    <InputOTP.Slot index={5} />
  </InputOTP.Group>
</InputOTP.Root>
```
