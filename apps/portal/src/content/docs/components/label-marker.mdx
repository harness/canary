## Usage

```typescript jsx
import { LabelMarker } from '@harnessio/ui/components'

return (
  <div className="inline-flex flex-wrap gap-10">
    <LabelMarker label="Bug" value="High Priority" color="red" inTable={false} />
    <LabelMarker label="Feature" value="New" color="blue" inTable={false} />
    <LabelMarker label="Improvement" value="Medium Priority" color="green" inTable={false} />
    <LabelMarker label="Task" value="Low Priority" color="yellow" inTable={false} />
    <LabelMarker label="Research" value="Optional" color="purple" inTable={false} />
  </div>
)
```

## Anatomy

The Label Marker component can be used to display labels with optional values and color coding.

```typescript jsx
<LabelMarker
  label="Bug"
  value="High Priority"
  color="red"
  inTable={false}
/>
```
