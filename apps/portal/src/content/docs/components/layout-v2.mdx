## Overview

The LayoutV2 component provides a set of flexible layout primitives for building consistent UI layouts. It includes Grid, Flex, Horizontal, and Vertical components with a clean, type-safe API.

## Flex Component

The Flex component is a versatile layout container that allows you to arrange items in a row or column with various alignment and spacing options.

## API Reference

### Flex Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `direction` | `'row' \| 'column' \| 'row-reverse' \| 'column-reverse'` | `'row'` | Direction of the flex container |
| `align` | `'start' \| 'center' \| 'end' \| 'baseline' \| 'stretch'` | - | Alignment of items on the cross axis |
| `justify` | `'start' \| 'center' \| 'end' \| 'between'` | - | Alignment of items on the main axis |
| `wrap` | `'nowrap' \| 'wrap' \| 'wrap-reverse'` | - | Whether items should wrap to the next line |
| `gap` | `string` | - | Space between items (using Tailwind spacing) |
| `gapX` | `string` | - | Horizontal space between items |
| `gapY` | `string` | - | Vertical space between items |
| `as` | `'div' \| 'span'` | `'div'` | HTML element to render |
| `className` | `string` | - | Additional CSS classes |

### Grid Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `columns` | `string` | - | Grid template columns |
| `rows` | `string` | - | Grid template rows |
| `flow` | `'row' \| 'column' \| 'dense' \| 'row-dense' \| 'column-dense'` | - | Grid auto flow |
| `align` | `'start' \| 'center' \| 'end' \| 'baseline' \| 'stretch'` | - | Alignment of items on the vertical axis |
| `justify` | `'start' \| 'center' \| 'end' \| 'between'` | - | Alignment of items on the horizontal axis |
| `gap` | `string` | - | Space between items (using Tailwind spacing) |
| `gapX` | `string` | - | Horizontal space between items |
| `gapY` | `string` | - | Vertical space between items |
| `as` | `'div' \| 'span'` | `'div'` | HTML element to render |
| `className` | `string` | - | Additional CSS classes |

### Horizontal Props

| Prop | Type | Default | Description |
| ---- | ---- | ------- | ----------- |
| `spacing` | `'small' \| 'medium' \| 'large'` | `'medium'` | Predefined spacing between items |
| `align` | `'start' \| 'center' \| 'end' \| 'baseline' \| 'stretch'` | - | Alignment of items on the cross axis |
| `justify` | `'start' \| 'center' \| 'end' \| 'between'` | - | Alignment of items on the main axis |
| `as` | `'div' \| 'span'` | `'div'` | HTML element to render |
| `className` | `string` | - | Additional CSS classes |

### Vertical Props

| ---- | ---- | ------- | ----------- |
