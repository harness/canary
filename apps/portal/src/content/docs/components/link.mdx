The `Link` component provides styled links for navigation within and outside the application. It offers two size options and distinguishes between internal and external links with appropriate visual indicator. The component ensures consistent styling while maintaining accessibility standards.

## Usage

```typescript jsx
import { Link } from '@harnessio/ui/components'

// ...

return (
  <>
    <Link href="/your-url-here">Username</Link>
  </>
)
```
