The `LogoV2` component contains logos of various brands and can be used as icons. It follows the brand guidelines to choose colors.

## Usage

```typescript jsx
<LogoV2 name="node-js" size="lg" />
```

<DocsPage.ComponentExample
  client:only
  code={`<LogoV2 name="node-js" size="lg" />`}
/>

## Sizes

The `LogoV2` component can be sized using the `size` prop.

<DocsPage.ComponentExample
  client:only
  code={`
  <Layout.Flex gap="md" justify="center" align="end" className="min-w-[400px]">
  <LogoV2 name="node-js" size="lg" />
  <LogoV2 name="node-js" size="md" />
  <LogoV2 name="node-js" size="default" />
  <LogoV2 name="node-js" size="sm" />
  <LogoV2 name="node-js" size="xs" />
  </Layout.Flex>
  `}
/>

## API Reference

```typescript jsx
<LogoV2
  name="logo-name"     // name of the logo
  size="default"       // [OPTIONAL] size variant of the logo
  className="my-class" // [OPTIONAL] custom class name
/>
```
