The Meter component provides a UI element for displaying progress or performance metrics. It includes a visual representation of the metric.

import { DocsPage } from "@/components/docs-page";

  client:only

      { id: '3', state: MeterState.Error },
    ]}
/>`}
/>

## Usage

```typescript jsx
import { Meter, MeterState } from '@harnessio/ui/components'

return (
  <Meter
    data={[
      { id: '1', state: MeterState.Success },
      { id: '2', state: MeterState.Warning },
      { id: '3', state: MeterState.Error },
    ]}
  />
)
```

## Anatomy

The Meter component can be used to display progress or performance metrics with a visual representation.

```typescript jsx
<Meter
  data={[
    { id: '1', state: MeterState.Success },
    { id: '2', state: MeterState.Warning },
    { id: '3', state: MeterState.Error },
  ]}
/>
```
