The `ModalDialog` component provides a modal dialog overlay for displaying important content or requesting user input. It includes features like different sizes, themes, icons/logos, and scrollable content areas.

import { DocsPage } from "@/components/docs-page";

## Components

### ModalDialog.Root

The root component that manages the modal's open state.

| Prop           | Type                      | Default     | Description                                          |
| -------------- | ------------------------- | ----------- | ---------------------------------------------------- |
| `open`         | `boolean`                 | `undefined` | Controls the open state when used in controlled mode |
| `onOpenChange` | `(open: boolean) => void` | `undefined` | Callback when open state changes                     |
| `children`     | `ReactNode`               | Required    | Trigger and Content components                       |

### ModalDialog.Content

The main content container of the modal.

| Prop        | Type                   | Default  | Description                      |
| ----------- | ---------------------- | -------- | -------------------------------- |
| `size`      | `"sm" \| "md" \| "lg"` | `"sm"`   | Controls the width of the modal  |
| `hideClose` | `boolean`              | `false`  | Hides the close button when true |
| `children`  | `ReactNode`            | Required | Modal content components         |

### ModalDialog.Header

The header section of the modal.

| Prop       | Type                                 | Default     | Description                      |
| ---------- | ------------------------------------ | ----------- | -------------------------------- |
| `icon`     | `string`                             | `undefined` | Name of icon to display          |
| `logo`     | `string`                             | `undefined` | Name of logo to display          |
| `theme`    | `"default" \| "warning" \| "danger"` | `"default"` | Theme for icon/logo color        |
| `children` | `ReactNode`                          | Required    | Title and Description components |

Note: `icon` and `logo` props cannot be used together.

### ModalDialog.Body

The main content area of the modal. Content is automatically scrollable.

| Prop        | Type        | Default  | Description                       |
| ----------- | ----------- | -------- | --------------------------------- |
| `className` | `string`    | `""`     | Class for the scrollable viewport |
| `children`  | `ReactNode` | Required | Modal content                     |

### ModalDialog.Footer

Container for modal actions, typically buttons.

| Prop        | Type        | Default  | Description        |
| ----------- | ----------- | -------- | ------------------ |
| `className` | `string`    | `""`     | Additional classes |
| `children`  | `ReactNode` | Required | Footer content     |

## Usage

```tsx
import { ModalDialog } from "@harnessio/ui";

<ModalDialog.Root open={isOpen} onOpenChange={setIsOpen}>
  <ModalDialog.Trigger asChild>
    <button>Open</button>
  </ModalDialog.Trigger>

  <ModalDialog.Content size="md" hideClose={false}>
    <ModalDialog.Header>
      <ModalDialog.Title>Heading</ModalDialog.Title>
      <ModalDialog.Description>Description text</ModalDialog.Description>
    </ModalDialog.Header>
    <ModalDialog.Body>Dialog content here.</ModalDialog.Body>
    <ModalDialog.Footer>
      <ModalDialog.Close>Cancel</ModalDialog.Close>
      <button className="btn btn-primary">Submit</button>
    </ModalDialog.Footer>
  </ModalDialog.Content>
</ModalDialog.Root>;
```
