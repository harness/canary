The More Actions Tooltip component provides a UI element for displaying a button that, when clicked, shows a tooltip with a list of actions.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<MoreActionsTooltip 
    actions={[
      { title: 'Edit', onClick: () => console.log('Edit clicked') },
      { title: 'Delete', onClick: () => console.log('Delete clicked'), isDanger: true }
    ]}
/>`}
/>

## Usage

```typescript jsx
import { MoreActionsTooltip } from '@harnessio/ui/components'

return (
  <MoreActionsTooltip
    actions={[
      { title: 'Edit', onClick: () => console.log('Edit clicked') },
      { title: 'Delete', onClick: () => console.log('Delete clicked'), isDanger: true }
    ]}
  />
)
```
