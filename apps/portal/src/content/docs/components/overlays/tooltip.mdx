The Tooltip component provides additional information for an element in a small overlay.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<TooltipProvider>
    <div className="flex justify-center">
        <Tooltip
            content="Tooltips provide helpful context and information"
        >
            <Button variant="secondary">Hover me</Button>
        </Tooltip>
    </div>
</TooltipProvider>`}
/>

## Usage

```typescript jsx
import { Tooltip, TooltipProvider } from '@harnessio/ui/components'

// ...

return (
  <TooltipProvider> {/* typically wrapped around the entire app */}
    {/* ... */}
    <Tooltip
        title="Title"
        content={<p>This is a tooltip</p>}
    >
      <span>Hover me</span>
    </Tooltip>
  </TooltipProvider>
)
```

## `withTooltip` HOC

HOC for adding Tooltip support to any component.
For the HOC to work correctly, the component must use forwardRef.

```typescript jsx
import { withTooltip } from "@harnessio/ui/components";

// ...

const comp = withTooltip(yourComp);
```

## Themes

The tooltip supports `default` and `themed` themes. The default theme provides high-contrast appearance with fixed colors and is used by default. The themed theme follows the current theme's color palette.

<DocsPage.ComponentExample
  client:only
  code={`
    <TooltipProvider>
      <ButtonLayout className="justify-center">
        <Tooltip
          content="High-contrast tooltip styling"
        >
          <Button variant="secondary">Default Theme</Button>
        </Tooltip>

        <Tooltip
          theme="themed"
          content="Follows current theme colors"
        >
          <Button variant="secondary">Themed</Button>
        </Tooltip>
      </ButtonLayout>
    </TooltipProvider>

`}
/>

## Vertical Scroll

Content scrolls when it exceeds the max height.

<DocsPage.ComponentExample
  client:only
  code={`
    <TooltipProvider>
      <div className="flex justify-center">
        <Tooltip
          title="Tooltip with scroll"
          content={
            <ul className="flex flex-col gap-y-cn-4xs">
              {Array.from({ length: 30 }, (_, i) => i + 1).map((i) => (
                <li key={i}>Item {i}</li>
              ))}
            </ul>
          }
        >
          <Button variant="secondary">Hover for scrollable list</Button>
        </Tooltip>
      </div>
    </TooltipProvider>
  `}
/>

## Horizontal Scroll

Content scrolls when it exceeds the max width.

<DocsPage.ComponentExample
  client:only
  code={`
    <TooltipProvider>
      <div className="flex justify-center">
        <Tooltip
          title="Tooltip with horizontal scroll"
          content={
            <div className="flex flex-col gap-y-cn-4xs">
              {Array.from({ length: 3 }, (_, row) => (
                <div key={row} className="flex gap-x-cn-md whitespace-nowrap">
                  {Array.from({ length: 15 }, (_, i) => row * 15 + i + 1).map((i) => (
                    <span key={i}>Item {i}</span>
                  ))}
                </div>
              ))}
            </div>
          }
        >
          <Button variant="secondary">Hover for scrollable list</Button>
        </Tooltip>
      </div>
    </TooltipProvider>
  `}
/>

## Footer

Use the `footer` prop for content that stays fixed at the bottom when the body scrolls.

<DocsPage.ComponentExample
  client:only
  code={`
    <TooltipProvider>
      <div className="flex justify-center">
        <Tooltip
          title="Tooltip with footer"
          content={
            <ul className="flex flex-col gap-y-cn-4xs">
              {Array.from({ length: 20 }, (_, i) => i + 1).map((i) => (
                <li key={i}>Item {i}</li>
              ))}
            </ul>
          }
          footer={
            <Button variant="link" size="md">
              View details
            </Button>
          }
        >
          <Button variant="secondary">Hover for tooltip with footer</Button>
        </Tooltip>
      </div>
    </TooltipProvider>
  `}
/>

## Sides

The tooltip can be positioned on any side: `top` (default), `bottom`, `left`, or `right`.

          side="bottom"

## Alignment

Control tooltip alignment relative to the trigger using `align` prop: `start`, `center` (default), or `end`.

<DocsPage.ComponentExample
  client:only
  code={`

      <ButtonLayout className="justify-center">
        <Tooltip
          side="top"
          content="Tooltip centered on the trigger"
        >
          <Button variant="secondary">Center (default)</Button>
        </Tooltip>
        <Tooltip
          side="top"
          align="start"
          content="Tooltip aligned to the start"
        >
          <Button variant="secondary">Start Aligned</Button>
        </Tooltip>
        <Tooltip
          side="top"
          align="end"
          content="Tooltip aligned to the end"
        >
          <Button variant="secondary">End Aligned Button</Button>
        </Tooltip>

    </TooltipProvider>
  `}
/>
