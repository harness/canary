The Progress component provides a UI element for displaying progress or performance metrics. It includes a visual representation of the metric.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<Progress value={75} max={100} />`}
/>

## Usage

```typescript jsx
import { Progress } from '@harnessio/ui/components'

return (
  <Progress value={75} max={100} />
)
```
