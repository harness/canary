The Radio component provides a UI element for selecting a single option from a list of options. It includes a radio button and a label.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<RadioGroup defaultValue="option1">
    <div className="flex items-center gap-4">
      <RadioButton value="option1" id="option1" />
      <Label htmlFor="option1">Option 1</Label>
    </div>
    <div className="flex items-center gap-4">
      <RadioButton value="option2" id="option2" />
      <Label htmlFor="option2">Option 2</Label>
    </div>
</RadioGroup>`}
/>

## Usage

```typescript jsx
import { RadioGroup, RadioButton, Label } from '@harnessio/ui/components'

return (
  <RadioGroup>
    <div className="flex items-center gap-4">
      <RadioButton value="option1" id="option1" />
      <Label htmlFor="option1">Option 1</Label>
    </div>
    <div className="flex items-center gap-4">
      <RadioButton value="option2" id="option2" />
      <Label htmlFor="option2">Option 2</Label>
    </div>
  </RadioGroup>
)
```

## Anatomy

The Radio component can be used to select a single option from a list of options.

```typescript jsx
<RadioGroup>
  <div>
    <RadioButton value="option1" id="option1" />
    <Label htmlFor="option1">Option 1</Label>
  </div>
  <div>
    <RadioButton value="option2" id="option2" />
    <Label htmlFor="option2">Option 2</Label>
  </div>
</RadioGroup>
```
