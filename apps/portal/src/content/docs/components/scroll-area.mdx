The Scroll Area component provides a customizable scrollable container with optional scrollbars. It is useful for displaying content that exceeds the available viewport space.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`
<div>
  <ScrollArea orientation="both" className="w-[300px] h-[200px] border">
      <div className="p-4 space-y-4 w-[700px] h-[300px]">
        <p>Content that exceeds the viewport height and width.</p>
        <p>More content...</p>
        <p>Even more content...</p>
      </div>
  </ScrollArea>
</div>`}
/>

## Usage

```typescript jsx
import { ScrollArea } from '@harnessio/ui/components';

//...

return (
  <ScrollArea orientation="both" className="w-[300px] h-[200px] border">
    <div className="p-4 space-y-4 w-[700px] h-[300px]">
      <p>Content that exceeds the viewport height and width.</p>
      <p>More content...</p>
      <p>Even more content...</p>
    </div>
  </ScrollArea>
);
```

### Directions

The `ScrollArea` component supports the `direction` prop to set the text direction. It accepts values like `ltr` (left-to-right) and `rtl` (right-to-left). The default value is `ltr`.

### Orientations

The `ScrollArea` component also supports the `orientation` prop to control the scroll direction. It accepts the following values: `horizontal`, `vertical`, and `both`. The default value is `vertical`.

        <p>More content...</p>
