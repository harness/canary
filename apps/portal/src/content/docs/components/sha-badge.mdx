The SHA Badge component provides a UI element for displaying a commit SHA with optional actions. It includes a root container, content area, and an icon button.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<div className="inline-flex"> 
    <ShaBadge.Root>
      <ShaBadge.Content>
        abc1234
      </ShaBadge.Content>
      <ShaBadge.Icon handleClick={() => console.log('Icon clicked')}>
        <Icon name="clone" size={16} />
      </ShaBadge.Icon>
    </ShaBadge.Root>
  </div>
  `}
/>

## Usage

```typescript jsx
import { ShaBadge } from '@harnessio/ui/components'

return (
  <ShaBadge.Root>
    <ShaBadge.Content>
      abc1234
    </ShaBadge.Content>
    <ShaBadge.Icon handleClick={() => console.log('Icon clicked')}>
      <Icon name="clone" size={16} />
    </ShaBadge.Icon>
  </ShaBadge.Root>
)
```

## Anatomy

The SHA Badge component can be used to display a commit SHA with optional actions.
