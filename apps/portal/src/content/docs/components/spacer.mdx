The `Spacer` component is used to add vertical spacing between elements.

<Aside>
  It is preferable to control spacing between elements using CSS layouts with
  flex or grid and to govern the spacing between elements with `gap`.
</Aside>

## Usage

```typescript jsx
import { Spacer } from '@harnessio/ui/components'

//...

return (
  <>
    <SomeComponent />
    <Spacer size={4} />
    <SomeOtherComponent />
  </>
)
```

## API Reference

The `Spacer` component can be used to add vertical spacing between elements. It accepts a `size` prop that specifies the
height of the spacer.

```typescript jsx
<Spacer
  size={4}          // [OPTIONAL] height of the spacer
  className="class" // [OPTIONAL] additional CSS classes to apply
/>
```
