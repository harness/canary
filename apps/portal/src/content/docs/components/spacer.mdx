The Spacer component provides a UI element for adding vertical spacing between elements. It includes various size options.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<>
    <div>Element 1</div>
    <Spacer size={4} />
    <div>Element 2</div>
</>`}
/>

## Usage

```typescript jsx
import { Spacer } from '@harnessio/ui/components'

return (
  <div>
    <div>Element 1</div>
    <Spacer size={4} />
    <div>Element 2</div>
  </div>
)
```
