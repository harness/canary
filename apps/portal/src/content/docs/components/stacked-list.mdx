The Stacked List component provides a UI element for displaying a list of items in a stacked layout. It includes various customization options.

import { DocsPage } from "@/components/docs-page";

      <StackedList.Field title="Item 2" description="Description for item 2" />
    </StackedList.Item>
  </StackedList.Root>`}
/>

## Usage

```typescript jsx
import { StackedList } from '@harnessio/ui/components'

return (
  <StackedList.Root>
    <StackedList.Item>
      <StackedList.Field title="Item 1" description="Description for item 1" />
    </StackedList.Item>
    <StackedList.Item thumbnail={<Icon name="clone" />} actions={<Button>Item Action</Button>} >
      <StackedList.Field title="Item 2" description="Description for item 2" />
    </StackedList.Item>
  </StackedList.Root>
)
```

## Anatomy

The Stacked List component can be used to display a list of items in a stacked layout with various customization options.

```typescript jsx
<StackedList.Root>
  <StackedList.Item>
    <StackedList.Field title="Item 1" description="Description for item 1" />
  </StackedList.Item>
  <StackedList.Item thumbnail={<Icon name="clone" />} actions={<Button>Item Action</Button>} >
    <StackedList.Field title="Item 2" description="Description for item 2" />
  </StackedList.Item>
</StackedList.Root>
```
