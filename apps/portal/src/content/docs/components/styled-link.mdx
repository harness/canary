The Styled Link component provides a UI element for creating styled links using `react-router-dom`. It includes optional features like custom class names.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<StyledLink to="/example">Example Link</StyledLink>`}
/>

## Usage

```typescript jsx
import { StyledLink } from '@harnessio/ui/components'

return (
  <StyledLink to="/example">Example Link</StyledLink>
)
```
