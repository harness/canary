The Switch component provides a UI element for toggling between two states. It includes a thumb that moves when the switch is toggled.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample client:only code={`<Switch defaultChecked />`} />

## Usage

```typescript jsx
import { Switch } from '@harnessio/ui/components'

return (
  <Switch defaultChecked />
)
```

## Anatomy

The Switch component can be used to toggle between two states.

```typescript jsx
<Switch defaultChecked />
```
