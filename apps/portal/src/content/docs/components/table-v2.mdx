## Usage

```typescript jsx
import { TableV2 } from '@harnessio/ui/components'

//...

return (
  <TableV2.Root variant="default">
    <TableV2.Header>
      <TableV2.Row>
        <TableV2.Head>Name</TableV2.Head>
        <TableV2.Head>Email</TableV2.Head>
        <TableV2.Head>Display Name</TableV2.Head>
        <TableV2.Head>Date added</TableV2.Head>
      </TableV2.Row>
    </TableV2.Header>
    <TableV2.Body>
      <TableV2.Row>
        <TableV2.Cell>John Doe</TableV2.Cell>
        <TableV2.Cell>john@example.com</TableV2.Cell>
        <TableV2.Cell>John</TableV2.Cell>
        <TableV2.Cell>2023-01-01</TableV2.Cell>
      </TableV2.Row>
    </TableV2.Body>
  </TableV2.Root>
)
```

## Anatomy

All parts of the `TableV2` component can be imported and composed as required.

```typescript jsx
<TableV2.Root>
  <TableV2.Caption />
  <TableV2.Header>
    <TableV2.Row>
      <TableV2.Head />
    </TableV2.Row>
  </TableV2.Header>
  <TableV2.Body>
    <TableV2.Row>
      <TableV2.Cell />
    </TableV2.Row>
  </TableV2.Body>
  <TableV2.Footer>
    <TableV2.Row>
      <TableV2.Cell />
    </TableV2.Row>
  </TableV2.Footer>
</TableV2.Root>
```

## API Reference

<Aside>
  All components extend their HTML equivalents, so all relevant HTML props also
  apply.
</Aside>

### `Root`

The root component for the table.

```typescript jsx
<TableV2.Root
  variant="default"                 // [OPTIONAL] Variant of the table
  disableXScroll={false}            // [OPTIONAL] Disable horizontal scroll
  className="custom-class"          // [OPTIONAL] Custom class name
  tableClassName="custom-table-class" // [OPTIONAL] Custom table class name
>
</TableV2.Root>
```
