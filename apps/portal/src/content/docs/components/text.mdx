The Text component provides a UI element for displaying text with various styles and properties. It includes options for alignment, size, weight, color, and more.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample

    <Separator className="my-4" />
    <Text size={2} weight="medium" color="secondary">Another Example Text</Text>
    <Separator className="my-4" />
    <Text size={3} weight="semibold">Yet Another Example Text</Text>
</>`}
/>

## Usage

```typescript jsx
import { Text } from '@harnessio/ui/components'

return (
  <>
    <Text size={4} weight="bold" color="primary">Example Text</Text>
    <hr className="my-4" />
    <Text size={2} weight="medium" color="secondary">Another Example Text</Text>
    <hr className="my-4" />
    <Text size={3} weight="semibold" color="tertiary">Yet Another Example Text</Text>
  </>
)
```
