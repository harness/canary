The Textarea component provides a UI element for entering multi-line text. It includes optional features like labels, captions, and error messages.

import { DocsPage } from "@/components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<Textarea 
    id="example-textarea" 
    label="Textarea Label" 
    placeholder="Enter text here" 
    caption="This is a caption" 
    optional 
    resizable 
  />`}
/>

## Usage

```typescript jsx
import { Textarea } from '@harnessio/ui/components'

return (
  <Textarea
    id="example-textarea"
    label="Textarea Label"
    placeholder="Enter text here"
    caption="This is a caption"
    error="This is an error message"
    optional
    resizable
  />
)
```
