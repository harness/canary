`toast` is a toast component that is used to display notifications to the user. It is handled by `Toaster` component, that is added in the root of the app.

To show a simple toast, use the `toast` function.

<DocsPage.ComponentExample
  client:only
  code={`
    <Button
      onClick={() =>
        toast({
          title: 'Toast title',
          description: Array.from({ length: Math.floor(Math.random() * 1000) + 1 })
            .map(() => 'a')
            .join(''),
          options: {
            duration: 3000,
            action: {
              label: 'Undo',
              onClick: () => {
                console.log('Undo')
              }
            }
          }
        })
      }
    >
      Show toast
    </Button>`}
/>

<Code
  code={`
    toast({
      title: 'Toast title',
      description: 'Toast description',
      options: {
      /**
       * If duration is not provided, 
       * the toast will be closed after 10 seconds by default.
       */
        duration: 5000,
        dismissible: true,
        action: {
          label: 'Undo',
          onClick: () => {
            console.log('Undo')
          }
        }
      }
    })
  `}
  lang="typescript"
/>

## Toast types

`toast` can be used to show different types of toasts.

<ol>
  <li>- Default toast (`toast`)</li>
  <li>- Success toast (`toast.success`)</li>
  <li>- Danger toast (`toast.danger`)</li>
  <li>- Info toast (`toast.info`)</li>
  <li>- Loading toast (`toast.loading`)</li>
  <li>- Promise toast (`toast.promise`)</li>
</ol>

<Card title="Points to Note">

<ol className='space-y-cn-md'>
  <li>Maximum number of toasts that can be displayed at a time is 3.</li>
  <li>- If <code>duration</code> is not provided, the toast will be closed after <strong>10 seconds</strong> by default.</li>
  <li>- Close button in toast will be hidden only if <code>dismissible</code> is set to false in <code>options</code>. By default, it is set to true.</li>
  <li>- Action button can be added to toast by providing <code>action</code> in <code>options</code>.</li>
  <li>- Clicking of action button will execute the callback function provided in <code>action</code> and will not close the toast by default. If you want to close the toast after clicking the action button, you can use <code>toast.dismiss</code> in the callback function.</li>
  <li>- Promise toast and Loading toast cannot be dismissed by the user, it will be closed after the promise is resolved or rejected (Promise toast) or when the work in progress task is completed (Loading toast).</li>
  <li>- Promise toast and Loading toast cannot display a <code>description</code></li>
</ol>
</Card>

### Loading toast

<Aside type="caution">
  Loading toast needs to be closed manually. use `toast.dismiss` to close the
  toast using the toast id.
</Aside>

<DocsPage.ComponentExample
  client:only
  code={`
    <Button
      onClick={() => {
        const _toastId = toast.loading({
          title: 'Loading data...',
          options: {
            action: {
              label: 'Close',
              onClick: () => {
                toast.dismiss(_toastId)
              }
            }
          }
        })
      }}
    >
      Show loading toast
    </Button>
        `}
/>

### Promise toast

Promise toast accepts a **promise** as its **first argument** and an **options object** as its **second argument**.

<Aside type="caution">
  Promise toast cannot be dismissed by the user, it will be closed after 7
  seconds when the promise is resolved or rejected.
</Aside>

<DocsPage.ComponentExample
  client:only
  code={`
() => (
  <Button
    onClick={() => {
      const _toastId = toast.promise(new Promise((resolve, reject) => {
        const random = Math.random()
        setTimeout(() => {
          if (random > 0.5) {
            resolve('Promise resolved')
          } else {
            reject('Promise rejected')
          }
        }, 5000)
      }), {
        loadingMessage: 'Promise in progress',
        successMessage: 'Promise resolved successfully',
        errorMessage: 'Promise rejected',
        options: {
          action: {
            label: 'Action',
            onClick: () => {
              console.log('Action')
              toast.dismiss(_toastId)
            },
          },
        },
      })
    }
    }
  >
    Show promise toast
  </Button>
)
`}
/>
