The `Toggle` component provides a customizable button that can be toggled between two states. It supports various styling options through variants and sizes.

import { DocsPage } from "../../../components/docs-page";

<DocsPage.ComponentExample
  client:only

    <Toggle>Default</Toggle>
  `}
/>

## Usage

```typescript jsx
import { Toggle } from '@harnessio/ui/components'

//...

return (
    <Toggle
        size="sm"
        variant="outline-secondary"
        rounded
        tooltipProps={{
            content: <p>This is a tooltip</p>
        }}
    >
        Toggle
    </Toggle>
)
```

## Variants

The `Toggle` component accepts a `variant` prop to control its appearance in active and inactive states. Supported values are: `outline-primary`, `outline-secondary`, `ghost-primary`, and `ghost-secondary`.

<DocsPage.ComponentExample
  client:only
  code={`
    <ButtonLayout orientation="vertical">
        <Toggle>outline-primary</Toggle>
        <Toggle variant="outline-secondary">outline-secondary</Toggle>
        <Toggle variant="ghost-primary">ghost-primary</Toggle>
        <Toggle variant="ghost-secondary">ghost-secondary</Toggle>
    </ButtonLayout>
`}
/>

## Sizes

The `Toggle` component accepts a `size` prop to control its overall dimensions. Supported values are: `default`, `sm`, and `xs`.

<DocsPage.ComponentExample
  client:only
  code={`
    <ButtonLayout orientation="vertical">
        <Toggle>Medium</Toggle>
        <Toggle size="sm">Small</Toggle>
        <Toggle size="xs">Extra Small</Toggle>
    </ButtonLayout>
`}
/>

## Rounded

The `Toggle` component accepts a `rounded` prop. When provided, the underlying button will have fully rounded corners.

<DocsPage.ComponentExample
  client:only
  code={`
    <Toggle rounded>Rounded</Toggle>
`}
/>

## Icon only

The `Toggle` component accepts a `iconOnly` prop. When this prop is set, the `Toggle` is styled to fit the dimensions of an icon, making it suitable for icon-only buttons.

<DocsPage.ComponentExample
  client:only
  code={`
    <Toggle iconOnly>
        <IconV2 name="plus" />
    </Toggle>
`}
/>

## With tooltip

The `Toggle` component accepts a `tooltipProps` prop. When provided, the `Toggle` will be wrapped in a `Tooltip` component. The `tooltipProps` object can include the following properties from `TooltipProps`: `title`, `content`, `side`, and `align`.

<DocsPage.ComponentExample
  client:only
  code={`
    <Toggle
      tooltipProps={{
        title: 'Title',
        content: <p>This is a tooltip</p>
      }}
    >
        With tooltip
    </Toggle>
`}
/>
