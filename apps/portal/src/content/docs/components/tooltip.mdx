The Tooltip component provides additional information for an element in a small overlay.

import { DocsPage } from "../../../components/docs-page";

<DocsPage.ComponentExample
  client:only
  code={`<TooltipProvider>
    <Tooltip
        title="Full-Stack Development"
        content="A comprehensive approach to building applications that covers both front-end and back-end technologies."
    >
        <Button theme="default">Trigger</Button>
    </Tooltip.Root>
</TooltipProvider>`}
/>

## Usage

```typescript jsx
import { Tooltip, TooltipProvider } from '@harnessio/ui/components'

// ...

return (
  <TooltipProvider> {/* typically wrapped around the entire app */}
    {/* ... */}
    <Tooltip
        title="Title"
        content={<p>This is a tooltip</p>}
    >
      <span>Hover me</span>
    </Tooltip>
  </TooltipProvider>
)
```

## Tooltip `side` configuration

The preferred side of the trigger to render against when open (e.g., `top`, `bottom`, `left`, `right`).

## Tooltip `align` configuration

Specifies the alignment of the tooltip in relation to the anchor or trigger component (e.g., `start`, `center`, `end`).

<DocsPage.ComponentExample
  client:only
  code={`

      <Tooltip
        title="Full-Stack Development"
        content="A comprehensive approach to building applications that covers both front-end and back-end technologies."
      >
        <Button theme="default">Center align</Button>
      </Tooltip>
      <Tooltip
        align="start"
        title="Full-Stack Development"
        content="A comprehensive approach to building applications that covers both front-end and back-end technologies."
      >
        <Button theme="default">Start align</Button>
      </Tooltip>
      <Tooltip
        align="end"
        title="Full-Stack Development"
        content="A comprehensive approach to building applications that covers both front-end and back-end technologies."
      >
        <Button theme="default">End align</Button>
      </Tooltip>
      </ButtonLayout>
    </TooltipProvider>
  `}
/>
