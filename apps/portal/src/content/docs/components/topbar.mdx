The Topbar component provides a UI element for displaying a top navigation bar. It includes various sections like left, center, and right.

import { DocsPage } from "@/components/docs-page";

      Right Action contents
    </Topbar.Right>
  </Topbar.Root>`}
/>

## Usage

```typescript jsx
import { Topbar } from '@harnessio/ui/components'

return (
  <Topbar.Root>
    <Topbar.Left>
      <Button>Left Action</Button>
    </Topbar.Left>
    <Topbar.Center>
      <Text>Center Content</Text>
    </Topbar.Center>
    <Topbar.Right>
      <Button>Right Action</Button>
    </Topbar.Right>
  </Topbar.Root>
)
```
