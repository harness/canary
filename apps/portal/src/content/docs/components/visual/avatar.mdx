The `Avatar` component provides a way to represent an image with a fallback for when the image cannot be loaded. It supports displaying initials derived from the `name` prop or an Avatar Icon if no `name` or `src` is provided.

## Usage

```typescript jsx
import { Avatar } from '@harnessio/ui'

// ...

return (
  <Avatar name="John Doe" src="/avatar.png" />
)
```

## Sizes

The `Avatar` component supports four sizes: `xs`, `sm`, `md`, and `lg`. The default size is `sm`.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex items-center gap-cn-xl">
    <Avatar name="John Doe" src="/avatar.png" size="xs" />
    <Avatar name="John Doe" src="/avatar.png" size="sm" />
    <Avatar name="John Doe" src="/avatar.png" size="md" />
    <Avatar name="John Doe" src="/avatar.png" size="lg" />
</div>`}
/>

## Shapes

The `Avatar` component supports two visual shapes: `md` and `rounded`.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex items-center gap-cn-xl">
    <Avatar name="John Doe" />
    <Avatar name="John Doe" rounded />
</div>`}
/>

## No initials

In case you want the fallback string to be without initials, pass the `noInitials` prop.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex items-center gap-cn-xl">
    <Avatar noInitials name="+22" size="lg" rounded />
    <Avatar noInitials name="+222" size="lg" rounded />
</div>`}
/>

## Custom Icon

You can display a custom icon instead of initials or the default user icon using the `icon` prop.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex items-center gap-cn-xl">
    <Avatar icon="account" size="lg" rounded />
    <Avatar icon="organizations" size="lg" rounded />
    <Avatar icon="folder" size="lg" rounded />
    <Avatar icon="git-branch" size="lg" rounded />
</div>`}
/>

## With Tooltip

Use `AvatarWithTooltip` to display a tooltip on hover.

<DocsPage.ComponentExample
  client:only
  code={`<div className="flex items-center gap-cn-xl">
    <AvatarWithTooltip name="John Doe" src="/avatar.png" size="lg" rounded />
    <AvatarWithTooltip name="Jane Smith" size="lg" rounded />
    <AvatarWithTooltip 
      icon="folder" 
      size="lg" 
      rounded 
      tooltipProps={{ content: "Project folder" }} 
    />
</div>`}
/>
