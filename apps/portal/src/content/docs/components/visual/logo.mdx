Logo components for displaying third-party brand logos. Available in two variants: `LogoV2` for full-color logos and `LogoV2Symbol` for monochrome symbols.

## LogoV2

Full-color brand logos with background. Use for prominent brand representation.

### Usage

```typescript jsx
import { LogoV2 } from "@harnessio/ui/components";

<LogoV2 name="docker" size="lg" />
```

## LogoV2Symbol

Monochrome logo symbols using `currentColor`. Use for subtle UI contexts or when brand colors would be too prominent.

### Usage

```typescript jsx
import { LogoV2Symbol } from "@harnessio/ui/components";

<LogoV2Symbol name="docker" size="lg" color="default" />
```

<DocsPage.ComponentExample
  client:only
  code={`
