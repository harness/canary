This is the <span style={{fontWeight: 600}}>reference palette</span> — raw LCH color primitives that the design system is built from. Each of the 15 color families has 14 shades (25–1000), authored in <span style={{fontWeight: 600}}>LCH color space</span> for perceptual uniformity.

These values are <span style={{fontWeight: 600}}>not used directly</span> in components. Instead, they serve as building blocks for <a href="/design-system/color-system">semantic tokens</a> — which map color intent (background, text, border, status) to specific shades per theme.

<span style={{ fontWeight: 600 }}>Use this page to:</span>- Browse shades and
inspect their Lightness, Chroma, and Hue values - Test color pairs for WCAG 2.1
contrast compliance (AA / AAA) - Find the right primitives before assigning them
to semantic roles
