## Background Colors

Background colors establish the visual hierarchy and surface elevation in your UI. All background utilities support alpha transparency via Tailwind's `/[opacity]` syntax.

### Base Backgrounds

The base background colors provide the foundational layering system.

<DocsPage.ComponentExample
  client:only
  code={`<Layout.Grid columns="1fr 1fr 1fr 1fr" gap="md" className="p-cn-md">


### Status Colors

Status colors communicate success, danger, warning, and neutral states. Each status color has three variants: primary, secondary, and outline.

<Aside type="caution">
  **Important:** Always pair status backgrounds with their corresponding text
  colors: - `bg-cn-success-primary` + `text-cn-success-primary` -
  `bg-cn-success-secondary` + `text-cn-success-secondary` -
  `bg-cn-success-outline` + `text-cn-success-outline` +
  `border-cn-success-outline`
</Aside>

## Border Radius

The design system provides border radius utilities for different component types.

### Border Radius Scale

| Class | Value | Usage |
|-------|-------|-------|
| `rounded-cn-none` | 0px | No rounding |
| `rounded-cn-xs` | 2px | Minimal rounding |
| `rounded-cn-sm` | 4px | Small components |
| `rounded-cn-md` | 8px | Medium components |
| `rounded-cn-lg` | 12px | Large components |
| `rounded-cn-xl` | 16px | Cards, panels |
| `rounded-cn-full` | 9999px | Pills, avatars |
