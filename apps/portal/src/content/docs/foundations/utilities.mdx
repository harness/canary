## Background Colors

Background colors establish the visual hierarchy and surface elevation in your UI. All background utilities support alpha transparency via Tailwind's `/[opacity]` syntax.

### Base Backgrounds

The base background colors provide the foundational layering system.

<DocsPage.ComponentExample
  client:only
  code={`<Layout.Grid columns="1fr 1fr 1fr 1fr" gap="md" className="p-cn-md">
  <div className="bg-cn-0 border border-cn-3 rounded-3 p-cn-md text-center">
    <Text variant="caption-normal" className="mb-cn-xs">bg-cn-0</Text>
    <Text variant="caption-light">Lightest background</Text>
  </div>
  <div className="bg-cn-1 border border-cn-3 rounded-3 p-cn-md text-center">
    <Text variant="caption-normal" className="mb-cn-xs">bg-cn-1</Text>
    <Text variant="caption-light">Primary background</Text>
  </div>
  <div className="bg-cn-2 border border-cn-3 rounded-3 p-cn-md text-center">
    <Text variant="caption-normal" className="mb-cn-xs">bg-cn-2</Text>
    <Text variant="caption-light">Secondary background</Text>
  </div>
  <div className="bg-cn-3 border border-cn-3 rounded-3 p-cn-md text-center">
    <Text variant="caption-normal" className="mb-cn-xs">bg-cn-3</Text>

  </div>
</Layout.Grid>`}
/>

**Usage:**

```tsx
<div className="bg-cn-1">Primary surface</div>
<div className="bg-cn-2">Raised surface</div>
<div className="bg-cn-0/50">Semi-transparent background</div>
```

### Status Colors

Status colors communicate success, danger, warning, and neutral states. Each status color has three variants: primary, secondary, and outline.

<Aside type="caution">
  **Important:** Always pair status backgrounds with their corresponding text
  colors: - `bg-cn-success-primary` + `text-cn-success-primary` -
  `bg-cn-success-secondary` + `text-cn-success-secondary` -
  `bg-cn-success-outline` + `text-cn-success-outline` +
  `border-cn-success-outline`
</Aside>

| Blue   | `bg-cn-blue-`   | primary, secondary, outline |
| Purple | `bg-cn-purple-` | primary, secondary, outline |
| Brown  | `bg-cn-brown-`  | primary, secondary, outline |
| Cyan   | `bg-cn-cyan-`   | primary, secondary, outline |
| Indigo | `bg-cn-indigo-` | primary, secondary, outline |
| Lime   | `bg-cn-lime-`   | primary, secondary, outline |

| Orange | `bg-cn-orange-` | primary, secondary, outline |

  </div>
  <div className="bg-cn-brand-outline border border-cn-brand rounded-3 p-cn-md text-center">
    <Text variant="body-strong" className="mb-cn-xs">Outline</Text>
    <Text variant="caption-normal">bg-cn-brand-outline</Text>
  </div>
</Layout.Grid>`}
/>

**Hover and Selected States:**

```tsx
// Brand button with hover state
<button className="bg-cn-brand-primary hover:bg-cn-brand-primary-hover">
  Click me
</button>

// Brand button with selected state
<button className="bg-cn-brand-primary-selected">
  Selected
</button>
```
